DruxtJS Menu

Provides a Menu Vue component to be used within a Druxt (DRUpal nuXT) project.

Get started

The DruxtMenu module adds a Vue.js Menu component system for your Decoupled Drupal (opens new window) and Nuxt.js (opens new window) application.

# DruxtMenu

The DruxtMenu component inteligentally loads in your Drupal menu using the built in JSON:API as well as the Drupal JSON:API Menu Items (opens new window) module.

<DruxtMenu name="main" :depth="1" />

# DruxtJS

DruxtJS is a framework to connect a Drupal (opens new window) JSON:API backend to a Nuxt.js (opens new window) application.

Find out more at https://druxtjs.org (opens new window)
