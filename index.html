Druxt.js Menu

Provides a Menu Vue component to be used within a Druxt (DRUpal nuXT) project.

Get started

The Druxt.js Menu module adds support for Drupal Menus to your Nuxt.js frontend.

Example:

<druxt-menu />

# Druxt.js

Druxt.js is a suite of modules to connect a Drupal JSON:API backend to a Nuxt.js frontend.

Find out more at http://druxtjs.org
