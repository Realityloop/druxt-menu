# get(app, name)

Get menu by name.

  • Fetches the menu items from the JSON:API endpoint.
  • Commits the menu items to the Vuex state object.
    Vuex Action => Mutates state propertyentities

Kind: action method of druxtMenu

Param Type Description
app object The Nuxt app context.
name string The Menu name.

Example

await this.$store.dispatch('druxtMenu/get', 'main')
1

# addEntities

Adds specified Drupal JSON:API Menu Items data to the Vuex state object.

Kind: mutation method of druxtMenu

Param Type Description
state State The Vuex State object.
entities object The Drupal JSON:API Menu Item entities.

Example

this.$store.commit('druxtMenu/addEntities', entities)
1

# ~getEntitiesByFilter : function

Get entities by filter.

Kind: inner property of druxtMenu

Param Type Description
filter function A filter() method compatible function.

Example

const items = this.$store.getters.getEntitiesByFilter(key => {
  return this.entities[key].attributes.menu_name === 'main'
})
