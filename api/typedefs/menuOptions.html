# MenuOptions : object

DruxtMenu options.

Kind: global typedef
See: JSON:API Menu Items (opens new window)
Properties

Name Type Default Description
[jsonApiMenuItems] boolean false Whether to use the Drupal JSON:API Menu Items module endpoint.

Example

{
  jsonApiMenuItems: true,
}
